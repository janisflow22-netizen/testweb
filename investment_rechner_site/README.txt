
Investment-Rechner – Deployment-Hinweise

Diese Seite ist ein reines, statisches HTML (index.html). Du kannst sie kostenfrei hosten – hier drei schnelle Wege:

1) Netlify Drop (am einfachsten, ohne Account):
   - https://app.netlify.com/drop öffnen
   - Die 'index.html' (oder den gesamten Ordner) auf die Seite ziehen
   - Fertig – du bekommst sofort eine https://…netlify.app/ URL

2) GitHub Pages (mit GitHub-Account):
   - Neues Repo anlegen (public), Datei 'index.html' hochladen
   - In Repo → Settings → Pages → 'Deploy from a branch', Branch 'main' und Root auswählen
   - URL: https://<dein-name>.github.io/<repo>/

3) Cloudflare Pages (Account benötigt, aber sehr simpel):
   - https://dash.cloudflare.com → Pages → Create application → "Upload assets"
   - Ordner mit der 'index.html' hochladen
   - URL: https://<dein-projekt>.pages.dev/

Wichtig: Auf dem iPhone bitte die Seite immer als https:// URL in Safari/Chrome öffnen (nicht nur als lokale Vorschau in „Dateien“).

Viel Spaß! :-)
